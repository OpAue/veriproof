# Veriproof

> **LLM 보급화 시대, 온라인 시험의 신뢰성 회복을 위한 실시간 모니터링 및 사후 검토 시스템**

## 1. 시스템 개요

본 플랫폼은 학생, 감독관, 교수 세 유형의 사용자를 위한 웹 기반 온라인 시험 환경을 제공하며, 다음과 같은 4단계 프로세스로 동작합니다.

1. **시험 개설 단계**
   - 교수가 시험 및 문항 등록 시 응시용 코드/QR 및 감독관 링크 자동 발급
2. **응시 단계**
   - 학생은 전체 화면 모드에서 시험 응시
   - 키 입력, 붙여넣기, 포커스 이탈 등 모든 행위를 서버로 실시간 전송
   - 환경 이탈 감지 시 화면 흐림 처리로 부정행위 차단
3. **감독 단계**
   - 수집된 이벤트를 가중치 기반으로 집계하여 학생별 '주목도 점수' 계산
   - 감독관 대시보드에서 점수 순으로 학생 카드 정렬 및 강조
   - 상위 위험군 학생 우선 확인 및 조치
4. **사후 검토 단계**
   - 시험 종료 후 교수는 학생의 답안 작성 과정을 키 입력 단위로 영상처럼 재생 가능
   - 자동 생성된 사후 리포트를 통해 의심 학생 및 시그널 분포 검토

---

## 2. 시스템 목적 및 필요성

- **LLM 대응 신뢰성 확보**: 기존의 복사/붙여넣기 방식의 부정행위를 기술적으로 차단하여 온라인 시험의 공정성 제고
- **행정 효율화**: 자필 시험으로의 회귀가 어려운 IT/코딩 과목에서 수기 채점의 비효율성을 제거
- **감독 한계 극복**: 특수 브라우저 우회 가능성 및 수십 명의 화면을 동시 감시해야 하는 인적 자원의 한계를 시스템적 데이터(주목도 점수)로 해결
